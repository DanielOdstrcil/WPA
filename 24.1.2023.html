JavaScript Základy

JavaScript Hodnoty

V JavaScriptu můžeme pracovat s různými druhy hodnot, včetně čísel, řetězců, boolean hodnot, polí, objektů atd.

JavaScript Literály

Literály jsou základní stavební kameny jazyka JavaScript. Zahrnují:

  • Číselné literály: Například 10, 3.14, 5e3.
  • Řetězcové literály: Text psaný mezi dvojitými nebo jednoduchými uvozovkami.
  • Boolean literály: True nebo False.
  • Null hodnota: Speciální hodnota, která znamená "nic".
  • Undefined hodnota: Hodnota proměnné, která nebyla definována.

JavaScript Proměnné

Proměnné jsou kontejnery pro ukládání datových hodnot. V JavaScriptu můžeme použít různé klíčové slova pro deklaraci proměnných, jako jsou var, let a const.

Přiřazení hodnoty proměnné se provádí pomocí operátoru =.

JavaScript Operátory

JavaScript obsahuje různé typy operátorů pro provádění různých operací, jako jsou aritmetické operátory (+, -, *, /, %), relační operátory (<, >, <=, >=, ==, !=) atd.

JavaScript Výrazy

V JavaScriptu můžeme použít výrazy k vyhodnocení různých operací. Výraz může obsahovat hodnoty, proměnné, operátory a další výrazy.

Porovnání Datových Typů v JavaScriptu

Typ Dat Příklad Popis
Číslo (Number) 42 Reprezentuje číselnou hodnotu, může být celé nebo desetinné.
Řetězec (String) 'Hello' Textový řetězec psaný mezi jednoduchými nebo dvojitými uvozovkami.
Boolean true Představuje hodnotu True nebo False.
Objekt (Object) { name: 'John', age: 30 } Komplexní datový typ, který může obsahovat různé hodnoty.
Pole (Array) [1, 2, 3] Seznam hodnot uložených v jedné proměnné.
Null null Speciální hodnota, která znamená "nic" nebo "žádnou hodnotu".
Undefined undefined Představuje hodnotu proměnné, která nebyla definována.
